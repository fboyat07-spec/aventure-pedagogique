# KidAI Learning App V2

AI-powered learning app for children with adaptive learning, ethical gamification, and parent analytics.

## Repo Structure
- frontend: React Native app
- backend: Node.js + Express API
- ai-engine: AI orchestration and prompt modules
- gamification: XP, badges, streaks, rewards
- analytics: event schema and tracking
- database: Firestore rules and indexes
- shared: shared types and constants
- docs: product spec, OpenAPI, wireframes
