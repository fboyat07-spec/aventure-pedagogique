# Frontend

React Native (Expo) scaffold for KidAI.

## Run (local)
1. Install Node.js and Expo CLI.
2. From frontend/ run:
   - npm install
   - npm run start

## Structure
- App.js: app entry
- src/navigation: navigation setup
- src/screens: screen placeholders
- src/services: API and Firebase placeholders
- src/components: UI primitives
